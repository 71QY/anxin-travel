# 安心出行 - 安装使用说明

## 环境要求

- **JDK**: OpenJDK 17.0.18 (Temurin 构建)
- **Maven**: 3.6+
- **MySQL**: 5.7+
- **Redis**: 6.0+

## 快速开始

### 1. 解压项目

```bash
unzip travel2.zip
cd travel
```

### 2. 配置数据库

创建 MySQL 数据库：

```sql
CREATE DATABASE anxin_travel DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

导入数据库脚本：

```bash
mysql -u root -p anxin_travel < database/anxin_travel.sql
```

### 3. 配置 Redis

确保 Redis 服务已启动（默认端口 6379）：

```bash
# Windows
redis-server.exe

# Linux/Mac
redis-server
```

### 4. 配置文件修改

编辑 `src/main/resources/application.yml`，修改以下配置：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/anxin_travel?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: your_username    # 修改为你的 MySQL 用户名
    password: your_password    # 修改为你的 MySQL 密码
  
  redis:
    host: localhost            # 修改为你的 Redis 地址
    port: 6379                 # 修改为你的 Redis 端口
    password:                  # 如果 Redis 有密码，在此填写

# 第三方 API 密钥配置
tencent:
  map:
    key: your_tencent_map_key  # 腾讯地图 API Key

aliyun:
  ai:
    api-key: your_dashscope_key  # 阿里云 DashScope API Key
```

**获取 API Key：**
- 腾讯地图 Key: https://lbs.qq.com/dev/console/key/manage
- 阿里云 DashScope Key: https://dashscope.console.aliyun.com/apiKey

### 5. 编译运行

```bash
# 清理并编译
mvn clean package

# 运行项目
mvn spring-boot:run
```

或者使用 IDE（IntelliJ IDEA）：
1. 打开项目根目录
2. 等待 Maven 依赖下载完成
3. 运行 `TravelApplication.java`

### 6. 访问应用

服务启动成功后，访问：

- **后端接口**: http://localhost:8080
- **WebSocket**: ws://localhost:8080/ws/agent

## 常见问题

### Q1: 端口被占用

修改 `application.yml` 中的端口：

```yaml
server:
  port: 8081  # 改为其他可用端口
```

### Q2: 配置未生效

执行彻底清理：

```bash
# 1. 停止服务
# 2. 删除 target 目录
rm -rf target

# 3. 重新编译运行
mvn clean spring-boot:run
```

### Q3: JDK 版本错误

确认使用 JDK 17：

```bash
java -version
# 应显示: openjdk version "17.0.18"
```

在 IDEA 中设置：
- File → Project Structure → Project SDK → 选择 JDK 17
- File → Settings → Build, Execution, Deployment → Compiler → Java Compiler → Target bytecode version: 17

### Q4: Redis 连接失败

检查 Redis 是否启动：

```bash
# Windows
redis-cli ping
# 应返回: PONG

# Linux/Mac
redis-cli ping
```
